# Bank of Starsector

A Starsector mod that adds the **Persean Banking Confederation (PBC)** — a powerful neutral faction with a complete banking system.

## Features

### Banking System
- **Loans**: 5 tiers from Emergency (50k) to Sovereign (5M) with varying interest rates and terms
- **Investments**: Savings accounts, government bonds, commodity futures, venture funds, and military contracts
- **Credit Score**: FICO-like scoring (300-850) that affects loan availability and interest rates
- **Interest Engine**: Dynamic rates influenced by sector wars and market conditions

### Debt Collection
- 4-phase escalation: warnings → restrictions → collection fleets → asset seizure
- Income garnishment on player colonies for defaulted loans
- Bankruptcy system with structured recovery period

### The Persean Banking Confederation
- Full faction with the Aurum star system (3 planets + orbital station)
- Powerful defensive navy with high-quality ships and officers
- Neutral diplomatic stance — factions avoid war with the PBC
- Custom fleet types: Security Details, Enforcement Fleets, Confederation Task Forces

### Mod Integration
- **Nexerelin**: Wars affect interest rates, faction diplomacy prevents conflicts with PBC
- **NexusUI**: Banking dashboard overlay with real-time portfolio tracking

## Installation

1. Download the latest release
2. Extract to your `Starsector/mods/` folder
3. Enable "Bank of Starsector" in the launcher

## Dependencies

- **LazyLib** (required)
- **Nexerelin** (optional)
- **NexusUI** (optional)
